Express.js is a backend framework for building full-stack web applications with JavaScript.

1. Initialize a node project. `npm init`
2. Install express. `npm i express`
3. Optional: Install Nodemon (as dev dependency). `npm i nodemon -D`
4. Create a server.js file. This will be the source of your backend application.
5. Add a nodemon dev script in your package.json. `nodemon server.js`
